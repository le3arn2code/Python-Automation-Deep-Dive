# Lab 26: Reading Excel Files (Optional)

## Objectives
- Understand how to read Excel files using Python libraries.
- Learn to manipulate and access Excel cell data.
- Gain familiarity with basic data extraction and transformation techniques from Excel files.

## Steps
1. Install `openpyxl` with `pip install openpyxl`.
2. Run `create_excel.py` to generate a sample Excel file (`mydata.xlsx`).
3. Run `read_excel.py` to read and process data from the Excel file.

## Conclusion
You created an Excel file with Python and successfully read values, iterated rows, and calculated sums. This forms the foundation for working with Excel automation in Python.
